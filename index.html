How to read this

Observation counts

Sourced from GBIF, filtered to a Northern Ireland bounding box plus a UK country filter. These counts reflect recorded observations, not population size — a fall can mean fewer organisms, fewer people out recording, or both. Treat a flagged decline as a prompt to look closer, not a population estimate.

GenBank sequences

Sourced from NCBI GenBank. This tracks global research and sequencing attention on a species, not field abundance — counts almost never fall, so we describe them as steady, accelerating, or stalled rather than rising or declining.

How alerts are set

Only complete calendar years feed the % change shown for observations — the current year-to-date is displayed on charts but excluded from alerts, so a partial year is never mistaken for a decline. Species with an invasive-species watch flag are alerted the moment any record appears at all.

Data refresh

An automated job re-fetches every figure on a schedule (see the repository's GitHub Actions workflow) and commits the result. Nothing needs to be run by hand.
